# 🚀 Multi-Stack Enterprise System

Sistema baseado em arquitetura de microserviços com múltiplas tecnologias.

## 🧠 Tecnologias

* PHP (Laravel) + MySql
* Node.js + React + MongoDB
* ASP.NET Core (C# e VB) + Sql Server
* Python (FastAPI) + Vue + PostgreSql
* Java (JSF) + Oracle
* WordPress + MySql
* Codeignite + MySql + SQL Server + MongoDB
* MySQL, MongoDB, SQL Server, PostgreSQL, Oracle

## 🧱 Arquitetura

* API Gateway (Nginx)
* Autenticação centralizada (JWT)
* Comunicação via REST + RabbitMQ
* Monitoramento com Prometheus + Grafana
* Microservices independentes

## 🚀 Como rodar

```bash
docker-compose up -d --build
```

## 🌐 Acessos

* Laravel: http://localhost/laravel
* Node: http://localhost/node
* Python: http://localhost/python
* Java: http://localhost/java

## 🔐 Autenticação

JWT centralizado via `/auth`

## ☁️ Deploy

Deploy automatizado via GitHub Actions + AWS EC2

## 📊 Monitoramento

* Grafana: http://localhost:3001
* RabbitMQ: http://localhost:15672
